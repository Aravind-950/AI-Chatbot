# 🤖 CareerBot AI

An advanced AI chatbot built using Flask and Gemini API.

## 🚀 Features

* AI Chatbot
* Voice Input
* Dark Mode
* Typing Animation
* Chat History
* Responsive UI

## 🛠 Technologies Used

* Python
* Flask
* Gemini API
* HTML
* CSS
* JavaScript

## ▶️ Run Project

```bash
pip install -r requirements.txt
python app.py
```
